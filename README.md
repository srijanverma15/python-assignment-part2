# Restaurant Order Management System (Part 2)

This assignment focuses on using Python data structures like lists and dictionaries.

## What I did

- Displayed menu grouped by categories (Starters, Mains, Desserts)
- Performed basic menu analysis like total items and pricing
- Built a simple cart system to add, remove and update items
- Calculated final bill with GST
- Managed inventory using deep copy and stock deduction
- Analysed daily sales data and identified best performing day

## Notes

Main learning was around working with nested dictionaries and lists. Cart logic and inventory handling took some effort but helped in understanding real-world use cases.
